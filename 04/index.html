Basically I created a class call Cat. Inside the cat it has a state machine that tells if the cat shows up or not. The cat also has a variable called countDown that goes down in every loop so that the cat can switch its status in a certain amount of time.

As now we know how to rotate a image, I also made the cats come out in different angles.
