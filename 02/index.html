Built with Processing and Processing.js

Instead of a chase game, I made it some thing more like a escape game. However, you will need to catch the coins to won points and time. (One of the coin is always flying away from you and it values more points!)

For the characters in this game, I made a class for the hero and the enemies. And I have a variable to count for the frames so that the characters can change for a certain frames. I also have a variable to store the lives of the hero. Every time it hit by the ghost, it loses one life.
